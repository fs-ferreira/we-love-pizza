# We Love Pizza
Micro SaaS for order management in a pizza restaurant. Projects were separated in git submodules.

Developed with:

- Frontend
    - nextJS 14
    - ReactJS
    - shadcn/ui
    - axios
    - sonner
- Mobile
    - React Native
    - Expo
    - axios
- Backend
    - nodeJS
    - axios
    - multer
    - JWT
    - Prisma
